## 💡 비용 절감 팁

### 1. 이메일 전송 대신 다른 방법 사용

**무료 대안**:
```
CSV 다운로드: 
- 부스 운영자 대시보드 → "CSV 다운로드" 클릭
- 브라우저에 자동 저장
- 무제한, 무료

백업 파일 다운로드:
- 관리자 대시보드 → "데이터 백업" 클릭
- 전체 데이터 JSON 형식
- 무제한, 무료

클라우드 저장소 연동:
- Google Drive API 연동 (무료 15GB)
- Dropbox API 연동 (무료 2GB)
- OneDrive API 연동 (무료 5GB)
```

### 2. 대기열 최적화

**불필요한 대기열 사용 제거**:
```
대기가 없는 부스:
- 대기열 시스템 사용하지 않기
- 방명록만 사용 (요청 수 50% 감소)

QR 코드 접속 유도:
- QR로 접속한 사람만 대기번호 발급
- 운영자 고정 기기는 대기번호 없음
```

### 3. 캐싱 활용

**브라우저 캐싱**:
```
운영자 대시보드:
- 페이지 닫지 말고 그대로 유지
- 필요할 때만 새로고침

통계 확인 주기 조절:
- 실시간 필요 없으면 5분마다 확인
- 자동 새로고침 대신 수동 클릭
```

### 4. 행사 종료 후 데이터 정리

**스토리지 절약**:
```
행사 완료 후:
1. CSV 다운로드
2. 백업 파일 저장
3. 참가자 명단 초기화 (선택사항)
4. 다음 행사까지 대기
```

## 📞 비용 관련 문의

### 자주 묻는 질문

**Q: 무료 범위를 초과하면 자동으로 청구되나요?**
A: 아니요! Cloudflare는 무료 플랜 초과 시 자동 업그레이드하지 않습니다. 수동으로 유료 플랜을 선택해야 합니다.

**Q: 갑자기 참가자가 많아지면 어떻게 하나요?**
A: 무료 범위 초과 시 요청이 거부될 수 있습니다. 하지만 일 100,000 요청은 충분히 큰 범위입니다.

**Q: 이메일 한도를 초과하면?**
A: SendGrid 무료 플랜 초과 시 이메일이 전송되지 않습니다. CSV 직접 다운로드를 사용하세요.

**Q: 비용을 모니터링할 수 있나요?**
A: 네! Cloudflare Dashboard에서 실시간으로 사용량을 확인할 수 있습니다.
